Dzongkha to Hindi Language Translator using Machine Learning Translation

This project develops a Neural Machine Translation (NMT) model for translating between Dzongkha (Bhutan’s official language) and Hindi, aiming to bridge communication gaps between these two languages. It uses an encoder-decoder architecture with an attention mechanism to improve translation accuracy and fluency. A key challenge is the scarcity of high-quality bilingual datasets for Dzongkha and Hindi, which the project addresses by curating its own data. The NMT system is integrated into a real-time chatbot for practical language conversion. The goal is to support cultural and economic exchange between Bhutan and India by making translation more inclusive and accessible.
